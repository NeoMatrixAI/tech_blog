**AIFAP (AI Finance Agent Platform)** 시스템 인프라 관련 기술 블로그 콘텐츠입니다. **Medium**, **Substack**, **X (Twitter)**에 발행됩니다.

이 저장소에는 AIFAP 자동매매 시스템 인프라를 문서화하기 위한 블로그 초안, 발행된 글, 관련 자료가 포함되어 있습니다.

> 현재 Bitget 거래소와 연동되어 있으며, Binance, Bybit 등 주요 거래소 지원 예정입니다.

## 📂 디렉토리 구조

```
tech_blog/
├── README.md              # 프로젝트 개요 (영문)
├── CLAUDE.md              # Claude AI 작성 가이드라인
├── docs/                  # 문서 번역본
│   └── README.ko.md       # 한글 README (이 파일)
├── drafts/                # 블로그 초안
│   ├── 00_master_draft.md # 시스템 레퍼런스 (발행 X)
│   ├── 01_xxx.md          # 영문 (발행용)
│   ├── 01_xxx_ko.md       # 한글 (검토용)
│   └── ...
├── published/             # 발행 완료 (영문만 보관)
├── specs/                 # API spec 경로 인덱스
├── images/                # 블로그 이미지
└── assets/                # 다이어그램 및 기타 자료
```

## 📝 블로그 카테고리

**시스템** - 인프라 중심의 지속적인 기술 블로그

| 카테고리 | 설명 |
|----------|------|
| **업데이트** | 버전 업데이트, 새 기능, 인프라 변경 |
| **이슈** | 장애 대응, 버그 수정, 성능 문제 |
| **삽질 기록** | 트러블슈팅, 시행착오, 교훈 |
| **기타** | 도구, 팁, 일반 기술 노트 |

---

## 🔄 작업 흐름

1. **주제 선정** → 카테고리 선택 (업데이트/이슈/삽질 기록/기타)
2. **초안 작성** → Claude가 `drafts/`에 영문 + 한글 초안 작성
3. **검토** → 사용자가 한글 버전 검토, 이미지 추가
4. **미리보기** → 마크다운 뷰어에서 영문 버전 미리보기
5. **발행** → Medium/Substack에 복사하여 발행
6. **보관** → 영문 버전을 `published/`로 이동
